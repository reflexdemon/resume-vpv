# resume-vpv

Online resume of Venkateswara VP - a seasoned technology leader with 18+ years of experience building enterprise solutions.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
  config/resume.json   # Resume data
  index.js             # Entry point
  index.html           # HTML template
  styles.css          # Styles
```

## Features

- Responsive design
- JSON-driven content
- Webpack-based build
